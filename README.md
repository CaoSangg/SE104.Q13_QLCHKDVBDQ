ĐỒ ÁN MÔN HỌC NHẬP MÔN CÔNG NGHỆ PHẦN MỀM 
1. Giới thiệu đề tài

Tên đề tài: Phần mềm Quản lý cửa hàng kinh doanh vàng bạc đá quý.

Mục tiêu: Xây dựng một phần mềm quản lý các hoạt động cho cửa hàng kinh doanh trang sức.

2. Công nghệ sử dụng
Ngôn ngữ: C# (.NET Core).

Giao diện: WPF (Windows Presentation Foundation).

Cơ sở dữ liệu: SQL Server.

Kiến trúc: Mô hình 3 lớp (3-Tier Architecture): GUI - BLL - DAL cùng với các DTO.

3. Hướng dẫn cài đặt & chạy ứng dụng

Bước 1: Khởi tạo Database
Mở và excute file .sql trong thư mục Database.

Bước 2: Cấu hình kết nối
Tìm file DataProvider.cs trong lớp DAL.

Thay đổi chuỗi connectionString bằng Server Name trên máy tính bản thân.

Bước 3: Build & Run
Nhấn F5 trong Visual Studio để biên dịch và chạy phần mềm.

4. Tài khoản đăng nhập hệ thống

Admin: tài khoản: admin / mật khẩu: 123.

Nhân viên: tài khoản: nhanvien / mật khẩu: 123.
