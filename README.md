ĐỒ ÁN MÔN HỌC NHẬP MÔN CÔNG NGHỆ PHẦN MỀM - SE104
1. Giới thiệu đề tài
Tên đề tài: Phần mềm Quản lý cửa hàng kinh doanh vàng bạc đá quý.
Mục tiêu: Xây dựng một phần mềm quản lý các hoạt động cho cửa hàng kinh doanh trang sức.
2. Công nghệ sử dụng
Ngôn ngữ: C# (.NET Core).
Giao diện: WPF (Windows Presentation Foundation).
Cơ sở dữ liệu: SQL Server.
Kiến trúc: Mô hình 3 lớp (3-Tier Architecture): GUI - BLL - DAL cùng với các DTO.
4. Hướng dẫn cài đặt & Chạy ứng dụng
Khởi tạo Database:
Mở file .sql trong thư mục Database và nhấn Execute để tạo cấu trúc bảng và dữ liệu mẫu.
Cấu hình kết nối:
Tìm file DataProvider.cs trong lớp DAL và thay đổi chuỗi connectionString bằng Server Name trên máy tính bản thân.
Build & Run:
Nhấn F5 trong Visual Studio để biên dịch và chạy phần mềm.
Tài khoản đăng nhập: admin/123 hoặc nhanvien/123
